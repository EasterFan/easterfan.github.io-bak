第一章 面向对象

1.面向对象

1.1 类与对象的概念

面向对象的优点很多，理解面向过程，面向对象就很好理解了。

面向过程：放（大象，冰箱）
面向对象：大象（放，冰箱）

这里大象就是一个对象，是一个实例。 面向对象中，类是对象的抽象集合，包是类的管理器，是类的集合。

关于命名： 包全部小写，包名要显示出功能（com.easter.zoo） 类首字母大写，抽象的描述对象（Cat）

1.2 创建一个类

package com.easter.zoo;

/**
 * 定义一个类：属性+方法
 */

public class Cat {
    // 成员属性
    String name;
    int age;
    double weight;
    String specie;

    // 成员方法
    public void eat(){
        System.out.print("猫吃鱼");
    }

    public void run(){
        System.out.print("猫跑步");
    }

}

1.3 实例化对象--一个普通的爱情故事

实例化对象要在main方法中操作 对象可以调用类的方法，给类的属性赋值

package com.easter.zoo;

public class CatInit {
    public static void main(String args[]){
        //实例化对象
        Cat katty = new Cat();

        // 对象调用类中的方法，并给变量赋值
        katty.eat();
        katty.run();
        katty.age = 18;
    }
}

可以看到，这里创建类和实例化类被写在两个java文件中，这遵循了设计模式中的单一职能原则。 即一个类只做一件事。 单一原则的好处是可以提高代码的复用性，重构时也方便。

举个例子吧，如果一个类承载了太多的功能，当其中一个功能变化时，很容易让整个程序崩溃。 就好像走路玩手机，开车打电话，当一个类（人）中承载了多个方法，很容易就发生事故辣。

对象的声明与实例化--对象的本质

Cat Tom = new Cat();

实例化后的对象中存储的是一个内存地址。对象的本质是一个内存地址 Cat Tom是对象的声明，表示在内存的栈中分配了一个空间，内容为null； new Cat()是对象初始化，表示在内存的堆中分配了一个空间，内容为一串内存地址； 怎样将堆栈连接起来呢？-->用= 表示将堆中的内存地址传递到栈中，实例化之后，Tom就是拿到了房间的钥匙，可以开始在这块内存上进行装修了。

比邻而居的Tom和Jerry---同类对象之间的关系

Cat Tom = new Cat();
Cat Jerry = new Cat();

两次实例化之后，Tom和Jerry成了邻居，他们的毛坯房大小/布局/摆设完全相同，不同的是，Tom和Jerry的钥匙不同（即内存地址不同），Tom家装修，Jerry家的装修不会受影响。两人各自装修，互不影响

Tom和Jerry同居之后--对象引用

Cat Tom = new Cat();
Cat Jerry = Tom;

后来啊，Jerry决定和Tom同居，Tom配了一把自己家的钥匙给Jerry，此时，Tom和Jerry有了相同的两把钥匙（相同的内存地址）；Jerry把墙刷成了紫色，Tom不喜欢，Tom把墙刷成了黑色，最后，后粉刷的Tom覆盖了Jerry，墙成了黑色。此时的Tom和Jerry拥有同一个内存地址，他们的动作都会影响最终的显示效果。

1.4 构造方法

构造方法的作用

构造方法是new关键字的好搭档，帮助配置对象初始化。

构造方法的写法-无参

 public Cat(){

    }

构造方法的写法-有参

public Cat(String name,int age,double weight,String specie){
        this.name = name;
        this.age = age;
        this.weight = weight;
        this.specie = specie;
    }

这四个赋值表示的是，将参数值赋给属性值 右边的参数值是构造函数括号里的四个参数，左边的属性值，是Cat类中定义的四个成员变量。 为什么要加this？ this表示当前对象，程序具有就近原则，如果不指定是当前对象的name属性，程序通过就近原则就会找到参数name，而不是属性name，可以说，我们引入this关键字，是为了解决属性名和参数名相同时，帮助程序绕远路找到属性名。

当前对象是谁呢？ 哪个对象调用了构造函数，这个对象就是this。 Cat Tom = new Cat("Tom",2,8.0,"纯正蓝猫")，Tom在new时调用了上面的有参构造函数，this指的就是Tom

构造方法的特点

feather of constractor

  1. 构造方法名必须与类名相同，且无返回值
  2. 构造方法不能由对象单独调用，必须配合new关键字
  3. 构造方法可以没有（系统会默认给你一个无参构造方法），可以有多个，他们之间互为重载
  4. 构造方法不可以被对象调用，却可以被构造方法调用（通过this()）--即构造方法之间相互调用，这句代码必须放在第一行。
